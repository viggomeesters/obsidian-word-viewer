# Security

Word Viewer treats Word documents as untrusted local packages.

- It does not execute macros, VBA, OLE objects, embedded packages, scripts, or external relationships.
- It does not upload files or make network requests.
- It does not use the system clipboard.
- It does not write back to the source document.
- It does not launch external viewers.

Report security issues through GitHub security advisories or a private issue with reproduction details.

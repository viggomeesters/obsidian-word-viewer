# Changelog

## 0.1.1

- Renamed the plugin id and repository metadata from `docx-viewer` / `obsidian-docx-viewer` to `word-viewer` / `obsidian-word-viewer` because the Community directory already has a `docx-viewer` plugin.
- Renamed the display name from `DOCX Viewer` to `Word Viewer` and hardened community checks for manifest display-name punctuation.

## 0.1.0

- Initial read-only `.docx` viewer.
- Added document outline, paragraphs, lists, tables, hyperlinks, comments, footnotes, endnotes, media metadata, relationship warnings, and package diagnostics.
- Added fixtures and smoke checks for malformed, encrypted, embedded-object, no-comments, simple, and large documents.
